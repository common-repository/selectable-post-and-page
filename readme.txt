== Description ==
Display your selected post and page.
A very simple plugin to add selectable post and page - for WordPress. 

= Features include =
* Easy to install
* Use for any sidebar.
* Get post(s) directly by a list of selected posts.
* Widget to display the post feature image. 
* It will be displayed at random as the featured image of any post without an image.

== Installation ==
1. Upload the 'selectable-post-and-page' folder to the '/wp-content/plugins/' directory.
2. Activate the "selectable-post-and-page" list plugin through the 'Plugins' menu in WordPress.
3. Go to 'Appearance' > 'Widgets' and place the widget into a sidebar to configure it.
